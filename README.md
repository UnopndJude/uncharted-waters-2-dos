# 대항해시대 2 웹 플레이어 (DOS 버전)
# Uncharted Waters 2 Web Player (DOS Edition)

**한글** — 브라우저에서 바로 플레이하는 대항해시대 2 (DOS 버전) 웹 에뮬레이터입니다.
설치 없이 웹페이지에 접속해서 본인이 보유한 게임 파일을 불러오면 실행됩니다.

**English** — A browser-based emulator frontend for *Uncharted Waters: New Horizons* (DOS edition).
No installation required — just open the site and load a game bundle you legally own.

## 플레이 사이트 / Play Here

https://unopndjude.github.io/uncharted-waters-2-dos/

## 플레이 방법 / How to Play

**한글**
1. 위 "플레이 사이트" 링크로 접속합니다.
2. 화면의 **"게임 번들 업로드"** 버튼을 클릭합니다.
3. 준비한 `.jsdos` 파일(또는 `.zip`)을 선택합니다.
4. 잠시 로딩 후 게임이 시작됩니다.

**English**
1. Open the **Play Here** link above.
2. Click the **"Upload Game Bundle"** button on the page.
3. Select your `.jsdos` (or `.zip`) file.
4. After a brief load, the game starts.

## 게임 번들 만들기 / Creating a Game Bundle

**한글**
DOS 원본 파일은 여러 파일로 구성돼 있어 브라우저에서 바로 실행할 수 없습니다.
**하나의 `.jsdos` 번들 파일**로 묶어야 합니다.

1. 본인의 DOS 게임 폴더를 준비합니다 (실행 파일, 데이터 파일 등 포함).
2. https://dos.zone/studio/ 에 접속합니다.
3. 게임 폴더를 업로드하고, 실행할 `.EXE` 또는 `.BAT`을 지정합니다.
4. "Save" 또는 "Download" 버튼으로 `.jsdos` 파일을 받습니다.
5. 이 파일을 플레이 사이트에서 업로드하면 됩니다.

**English**
The original DOS release consists of multiple files and cannot run in a browser directly.
You need to bundle it as a single **`.jsdos` file**.

1. Prepare your DOS game folder (executables, data files, etc.).
2. Go to https://dos.zone/studio/.
3. Upload the folder and specify the `.EXE` or `.BAT` to launch.
4. Save/download the resulting `.jsdos` file.
5. Upload that file on the play site.

## 조작 방법 / Controls

**한글**
- **키보드**: 원작 DOS 조작을 그대로 사용합니다 (방향키, Enter, ESC, 스페이스 등).
- **마우스**: 일부 메뉴 선택에 사용됩니다.
- **전체 화면**: 에뮬레이터 UI의 풀스크린 버튼을 누르세요.
- **속도 조절**: 기본 속도가 빠르게 느껴지면 에뮬레이터 설정에서 CPU 사이클을 조정하세요.

**English**
- **Keyboard**: Original DOS controls (arrow keys, Enter, ESC, Space, etc.).
- **Mouse**: Used for some menu interactions.
- **Fullscreen**: Use the fullscreen button in the emulator UI.
- **Speed**: If the game runs too fast, adjust CPU cycles in the emulator settings.

## 저장 / Save Data

**한글**
- 게임 내 저장 기능을 사용하면 **브라우저에 자동으로 저장**됩니다.
- **저장 데이터는 브라우저별, 기기별로 분리**됩니다. 다른 브라우저/다른 PC에서는 불러올 수 없습니다.
- 브라우저 데이터를 삭제하거나 시크릿/프라이빗 모드로 접속하면 세이브가 날아갈 수 있습니다.
- 안전을 위해 주기적으로 세이브 폴더를 내보내 두세요.

**English**
- In-game saves are **stored in your browser automatically**.
- Save data is **isolated per browser and per device**. It will not transfer across browsers or machines.
- Clearing browser data or using incognito/private mode may lose your saves.
- Export save files periodically for safety.

## 자주 묻는 질문 / FAQ

**한글**

Q. 모바일에서도 되나요?  
A. DOS 버전은 키보드/마우스 조작이라 모바일에서는 불편합니다. SNES 버전을 추천드립니다 — 아래 "다른 버전" 참고.

Q. 한글 패치된 버전도 되나요?  
A. 네, 본인이 한글 패치된 DOS 파일을 가지고 있다면 그대로 번들로 만들어 사용할 수 있습니다.

Q. 음악이 안 나옵니다.  
A. 브라우저의 사운드 자동재생이 차단됐을 수 있습니다. 페이지를 클릭해 활성화하거나 브라우저 설정을 확인하세요.

Q. 화면이 너무 작아요.  
A. 에뮬레이터 UI의 전체 화면 버튼을 사용하거나, 브라우저 확대(Ctrl/Cmd + +)를 이용하세요.

Q. 게임이 느리거나 끊깁니다.  
A. 다른 탭을 닫고, 하드웨어 가속이 켜진 브라우저(Chrome, Edge, Firefox 최신)를 사용하세요.

**English**

Q. Does it work on mobile?  
A. DOS relies on keyboard/mouse, so it's uncomfortable on mobile. The SNES edition is recommended — see "Other Editions" below.

Q. Can I use a translated version?  
A. Yes. If you have a translated DOS copy, bundle it yourself and use it.

Q. No sound.  
A. Browser autoplay may be blocked. Click the page once to unlock audio, or check browser settings.

Q. Screen too small.  
A. Use the fullscreen button in the emulator UI, or zoom in (Ctrl/Cmd + +).

Q. The game is slow or stuttering.  
A. Close other tabs and use a recent browser (Chrome, Edge, Firefox) with hardware acceleration enabled.
